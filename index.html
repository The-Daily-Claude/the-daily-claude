The Daily Claude

This public repo contains The Trawl, the CLI behind The Daily Claude, plus the code, prompts, tests, and docs around it. The wider project turns selected AI-session excerpts into short entries after review and anonymisation.

Session logs are easy to lose and hard to reread. The Trawl is a rough filter for finding material worth a second look, not a finished editorial system.

View the Trawl Browse the repo

What this repo actually has

The public engineering work, and now a small set of example entries. Not a full public archive yet.

Privacy

We try to reduce risk by replacing or removing names, orgs, paths, credentials, and other identifying details before publication.

Known limits

The project uses verbatim quotes, and much of the anonymisation still relies on LLM judgement plus review. That lowers risk, but it does not make publication risk-free.

Current capabilities

Today, The Trawl scans local Claude Code logs, surfaces candidate excerpts, tokenises flagged entities, and writes markdown drafts for review.

Outside help

Help is most useful on privacy review, privacy hardening, future improvements, promotion, and general ideas and suggestions.

Near-term goals

Improve extraction quality, make review and anonymisation more reliable, and support more distributed contribution and review.

Later

Make it easier for people to share entries with us intentionally, and eventually publish a broader public site with more selected entries.
